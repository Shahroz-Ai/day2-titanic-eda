# 🚢 Titanic EDA - Exploratory Data Analysis

Day 2 of my 30-Day AI Challenge

## 📊 About
Exploratory Data Analysis on the Titanic dataset to uncover survival patterns.

## 🔍 Key Insights
- Overall survival rate: 38.2%
- Female survival rate: 74.0% vs Male: 18.9%
- 1st Class survival: 62.6% vs 3rd Class: 24.2%
- "Women and children first" was real!

## 🛠️ Tech Stack
- Python, Pandas, Matplotlib, Seaborn
